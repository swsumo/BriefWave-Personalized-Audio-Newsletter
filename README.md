# 🎙️ BriefWave — AI-Powered Audio News Briefing

 Your personalized India news. Generated by AI. Delivered as a podcast.

BriefWave is a fully automated AI pipeline that transforms real news headlines into a structured, radio-style audio briefing — tailored to your interests.

Built using **n8n**, **Google Gemini 2.5 Flash**, **GNews API**, **ElevenLabs**, and **Supabase** — with zero traditional backend code.

## 🚀 What It Does

* Fetches real headlines (India, yesterday)
* Filters by user-selected categories & sub-interests
* Generates structured radio broadcast script
* Converts to natural AI voice (MP3)
* Stores transcript + audio in Supabase
* Plays instantly in browser

---

## 🏗️ Architecture

```
Frontend → n8n Webhook
        → GNews API
        → Gemini (Script Generation)
        → ElevenLabs (TTS)
        → Supabase (DB + Storage)
        → Frontend Player
```

---

## 🛠 Tech Stack

* **Orchestration:** n8n
* **Summarization:** Gemini 2.5 Flash
* **News Source:** GNews API
* **TTS:** ElevenLabs
* **Database & Storage:** Supabase
* **Frontend:** HTML / CSS / JS

---

## 💡 Why This Project Is Strong

* API chaining
* Prompt engineering with constraints
* AI grounding (no hallucinated news)
* Binary file handling in n8n
* Cloud storage integration
* Full-stack AI system without backend server

---

## ⚠️ Limitations

* Free-tier API limits
* Manual trigger only
* Demo-level deployment
